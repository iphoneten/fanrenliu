# 凡人流文字修仙游戏

当前版本已从早期自由原型，重构为基于 [doc/凡人修仙游戏策划总稿v2.0.docx](/Users/andyfan/Desktop/pythonPro/fanren/doc/凡人修仙游戏策划总稿v2.0.docx) 的第一卷骨架版。

## 已按策划文档落地的核心点

- 废柴开局：主角默认偏低资质，灵根为伪灵根倾向
- 寿元压力：年龄与寿元大限成为主线资源
- 金手指盲盒：造化之瓶 / 混沌石珠 / 衍天之镜随机开局
- 第一卷节点地图：青石村 -> 无名后山 -> 凡俗集镇 -> 迷雾黑原 -> 太南谷
- 修炼公式倾向：修炼速度受主灵根纯度、灵根数量、悟性与金手指加成影响
- 炼丹雏形：黄龙丹、清灵散、丹毒、丹道熟练度
- 突破门槛：小瓶颈、丹毒惩罚、筑基丹需求预埋
- 背包与装备：Tab 背包、格子展示、自动拾取装备

## 当前前端结构

- `index.html`：主界面，已切换为 GDD 导向布局
- `styles.css`：三端响应式样式
- `src/game-core.js`：核心数值、节点流程、寿元、金手指、炼丹与突破
- `src/app.js`：界面渲染、交互、存档
- `doc/凡人修仙游戏策划总稿v2.0.docx`：原始策划文档
